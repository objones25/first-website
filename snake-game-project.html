Snake Game with AI

Overview

A Python-based implementation of the classic Snake game featuring multiple AI algorithms that can play autonomously. This project combines game development with artificial intelligence, showcasing different approaches to solving the Snake game challenge.

Key Features

  • Multiple AI algorithms including A* Pathfinding, Hamiltonian Cycle, and Greedy Algorithm
  • Classic snake gameplay mechanics
  • Multiple visual themes (Classic, Desert, Forest, Neon)
  • Adjustable game speed
  • Pause and resume functionality

Technical Details

Built using Python and Pygame, the project emphasizes object-oriented design with separate classes handling different aspects of the game. The AI implementations showcase various approaches to pathfinding and decision-making in a constrained environment.
