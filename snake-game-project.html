Snake Game Project

Inspiration

This project began as part of App Brewery's 100 Days of Code Python course, where I built a basic Snake game. While the course provided a solid foundation, I wanted to expand upon it significantly. I was particularly inspired by Code Bullet's video on implementing AI for Snake, which demonstrated various algorithms for autonomous gameplay. This led me to rebuild the game from scratch, adding multiple AI algorithms, enhanced graphics, and additional features to create a more comprehensive and educational project.

Project Overview

This is a Python-based implementation of the classic Snake game, built using Pygame. The project features multiple game modes, including traditional player-controlled gameplay and various AI algorithms that can play the game autonomously.

Key Features

  • Classic snake gameplay mechanics
  • Multiple AI algorithms:
    • A* Pathfinding
    • Hamiltonian Cycle
    • Greedy Algorithm
  • Different visual themes:
    • Classic
    • Desert
    • Forest
    • Neon
  • Adjustable game speed
  • Pause and resume functionality

Technical Implementation

The game is structured using object-oriented programming principles, with separate classes handling different aspects of the game:

  • SnakeGame: Main game logic and rendering
  • SnakeAI: Implementation of different AI algorithms
  • ThemeManager: Handles visual theming
  • Constants: Game configuration and enums

AI Algorithms

The project includes three different AI approaches to solving the Snake game:

  • A* Pathfinding: Uses heuristic search to find the optimal path to the food while avoiding obstacles
  • Hamiltonian Cycle: Creates a fixed path that visits every cell exactly once, guaranteeing a solution but not always the most efficient
  • Greedy Algorithm: Always moves toward the food using the shortest possible path, without considering long-term consequences

Try It Yourself

To run the game locally:

  1. Clone the repository from GitHub
  2. Install the required dependencies: pygame
  3. Run python main.py

Future Plans

I plan to enhance the game with additional features such as:

  • Online leaderboard system
  • Additional AI algorithms
  • More visual themes and customization options
  • Sound effects and background music
