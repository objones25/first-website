Cryptographic Library Project

Inspiration

My passion for cryptography was ignited during Math 116 at UCLA, where I studied both classical and modern encryption methods. This project represents an effort to deepen my understanding of cryptographic principles while creating a practical, educational library. The work also aligns with my goal of transitioning into cybersecurity.

Project Overview

This C++ cryptographic library, currently in active development, aims to provide a comprehensive suite of cryptographic tools. The project emphasizes security, performance, and educational value by implementing both historical and modern cryptographic algorithms.

Current Features

  • Modern Symmetric Encryption
    • AES implementation (128/192/256-bit) with hardware acceleration (AES-NI)
    • Multiple block cipher modes (ECB, CBC, CFB, OFB)
    • PKCS7 padding scheme
  • Hash Functions
    • SHA-256 implementation
    • HMAC functionality
  • Classical Ciphers (for educational purposes)
    • Caesar Cipher
    • Vigenère Cipher

Development Status

Currently working on resolving several key issues:

  • Fixing decryption round key transformation in AES-NI implementation
  • Correcting round key copy mismatches in key schedule
  • Debugging the AES-128 key schedule, particularly for decryption keys
  • Optimizing overall performance and memory usage
  • Enhancing test coverage

Planned Features

  • RSA implementation for asymmetric encryption
  • Digital signature functionality
  • Secure key management system
  • Additional hash functions (SHA-3)
  • Elliptic curve cryptography support
  • Advanced mathematical utilities for cryptographic operations

Technical Details

The library leverages modern C++ features while maintaining high security standards:

  • Written in C++17
  • Hardware acceleration using AES-NI where available
  • Extensive unit testing and benchmarking
  • Memory-safe implementations
  • Thread-safe operations

This project has been an incredible learning experience, allowing me to apply theoretical cryptographic knowledge to practical implementation challenges. I'm excited to continue expanding its capabilities and making it a valuable tool for both educational and practical purposes.
